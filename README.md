## 🎓 Hướng dẫn cho người mới

### 1. Cài đặt Node.js (Bắt buộc)

**Windows:**
1. Truy cập [nodejs.org](https://nodejs.org/)
2. Download bản LTS (Long Term Support)
3. Chạy file cài đặt và làm theo hướng dẫn
4. Khởi động lại terminal/command prompt

**macOS:**
```bash
# Sử dụng Homebrew
brew install node
```

**Linux (Ubuntu/Debian):**
```bash
# Cập nhật package list
sudo apt update

# Cài đặt Node.js
sudo apt install nodejs npm
```

### 2. Cài đặt Git (Bắt buộc)

**Windows:**
- Download tại [git-scm.com](https://git-scm.com/)
- Chạy file cài đặt

**macOS:**
```bash
brew install git
```

**Linux:**
```bash
sudo apt install git
```

### 3. Clone và chạy project

```bash
# 1. Mở terminal/command prompt/PowerShell
# 2. Di chuyển đến thư mục bạn muốn lưu project
cd Desktop

# 3. Clone project
git clone https://github.com/yourusername/gartic-clone.git

# 4. Vào thư mục project
cd gartic-clone

# 5. Cài đặt dependencies (chỉ chạy 1 lần đầu)
npm install

# 6. Chạy development server
npm run dev

# 7. Mở browser và truy cập http://localhost:8080
```

### 4. Sửa code và xem thay đổi

1. Mở project bằng VS Code hoặc editor yêu thích
2. Sửa file bất kỳ trong thư mục `src/`
3. Lưu file (Ctrl+S / Cmd+S)
4. Browser sẽ tự động reload và hiển thị thay đổi!

### 5. Các lỗi thường gặp và cách khắc phục

**Lỗi: "npm: command not found"**
```bash
# Kiểm tra Node.js đã cài chưa
node --version

# Nếu chưa có, cài đặt Node.js theo hướng dẫn trên
```

**Lỗi: "Cannot find module"**
```bash
# Xóa node_modules và cài lại
rm -rf node_modules
npm install
```

**Lỗi: "Port 8080 is already in use"**
```bash
# Đổi port trong vite.config.js
# Tìm dòng: port: 8080
# Đổi thành: port: 3000 (hoặc port khác)
```

**Lỗi: "EACCES: permission denied"**
```bash
# Linux/macOS - chạy với sudo
sudo npm install
```

## 🎨 Tùy chỉnh Project

### Đổi màu sắc chủ đạo

Sửa file `src/index.css`:

```css
:root {
  --primary: 210 100% 60%;      /* Màu xanh chủ đạo */
  --secondary: 45 100% 50%;     /* Màu vàng phụ */
  --success: 142 76% 36%;       /* Màu xanh lá */
  --danger: 0 84% 60%;          /* Màu đỏ */
}
```

### Thêm tính năng mới

1. Tạo component mới trong `src/components/`
2. Import và sử dụng trong pages
3. Thêm route mới trong `src/App.jsx` nếu cần

### Kết nối Backend

Project này hiện chỉ là frontend. Để thêm backend:

1. Cài đặt Socket.io cho realtime communication
2. Tạo server Node.js/Express
3. Kết nối WebSocket để đồng bộ vẽ và chat
4. Thêm database (MongoDB/PostgreSQL) để lưu điểm số
